 يجمع "نور الإسلام" بين التصميم العصري الأنيق وسهولة الاستخدام، ليوفر تجربة روحية هادئة ومنظمة.
أبرز المميزات:
خطة الأذكار الذكية: تصفح منظم للأذكار والتحصين اليومي مع عداد تفاعلي لكل ذكر.
المسبحة الإلكترونية المطورة: واجهة تسبيح احترافية تدعم الأهداف المخصصة والاهتزاز والمؤثرات البصرية.
مواقيت الصلاة الدقيقة: عرض حي لمواقيت الصلاة بناءً على موقعك مع مؤشر زمني للصلاة القادمة.
حكم وكنوز إيمانية: قسم متجدد للوعظ والرقائق لاستلهام الحكمة في كل وقت.
التقويم الهجري والميلادي: عرض مدمج للأيام والمناسبات الإسلامية الهامة.
تصميم فائق الدقة: واجهة مستخدم متجاوبة تماماً مع الوضع الليلي لضمان راحة العين أثناء القراءة.
